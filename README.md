Pesquisa sobre a Linguagem Python
 Sobre o projeto

Este repositório foi criado para a disciplina acadêmica com o objetivo de realizar uma pesquisa sobre a linguagem de programação **Python**, utilizando organização de tarefas inspirada em metodologia ágil.

O trabalho aborda informações sobre a linguagem, sua história, características, aplicações e exemplos práticos.

Integrante
- Victor Lucas de Araujo Silva

 Linguagem escolhida

**Python**

Python é uma linguagem de programação de alto nível, interpretada e multiparadigma, amplamente utilizada em desenvolvimento web, automação, ciência de dados, inteligência artificial e diversas outras áreas.
